# Game Boy Emulator in Rust

User must change the code in order to change which rom is loaded as of right now.

### Controls
|Emulator|Gameboy|
|---|---|
|W|Up|
|A|Left|
|S|Down|
|D|Right|
|K|A|
|M|B|
|H|Select|
|J|Start|
